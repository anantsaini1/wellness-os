## ✨ Features

| Feature | Description |
|---|---|
| 🍎 **Food & Calorie Tracking** | Log meals, track macros (protein, carbs, fat) with BMR/TDEE calculator |
| 😊 **Mood Logging** | Track emotional patterns with 30-day trend graphs |
| 😴 **Sleep Tracking** | Monitor sleep with quality ratings and goals |
| 💧 **Water Intake** | Visual water tracking with daily goals |
| 🧘 **Breathing Exercises** | 4-7-8, Box breathing, and Quick 2-4 modes |
| ✅ **Habit Tracking** | Physical, Mental & Family habit categories with streaks |
| 📊 **Stats Dashboard** | 30-day score, mood trends, streaks, achievements |
| 🔒 **PIN Lock** | Private data protection with 4-digit PIN |
| 🎨 **Themes** | Dark/light themes with 7 accent color options |
| 💾 **Backup & Export** | Export/import your data, generate monthly reports |

## 🚀 Getting Started

No installation needed — it runs entirely in the browser.

1. Visit the **[live demo](https://anantsaini1.github.io/wellness-os/)**
2. Start logging your meals, mood, and sleep
3. Check your stats to spot patterns

## 💡 Why I Built This

As a medical student, I understand how lifestyle — sleep, nutrition, and mental state — directly impacts health. I wanted a simple, unified place to track all three without juggling multiple apps.

This is my first project and I built it as a complete beginner using Claude AI as my coding companion.
